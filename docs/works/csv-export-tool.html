Demo

デモ動画（Before → After）

Problem

こんな作業、続いていませんか？

  • ECの検索結果を1件ずつコピペしてスプレッドシートへ
  • 毎週同じ一覧ページから価格・在庫をチェック
  • 転記ミスが出る・時間が取れない
Before 2分/件 手作業コピペ
→
After 10秒 ワンクリックCSV
Solution

このデモでできること

  • 一覧ページの商品情報を一括抽出
  • Excelで開けるCSV（UTF-8 BOM付き）
  • Manifest V3 対応のChrome拡張
  • ご依頼サイトのHTML構造に合わせてカスタマイズ可能

※ デモは架空ECページでの動作です。実在サイトは規約確認のうえ設計します。

How it works

使い方（3ステップ）

1

対象の一覧ページを開く

商品検索結果やリサーチ対象のリストページをブラウザで表示します。

2

拡張アイコンをクリック

検出件数とプレビューが表示されます。抽出ルールはサイトに合わせて調整可能です。

3

CSVをダウンロードしてExcelで開く

商品名・価格・在庫・カテゴリ・URLが列として出力されます。
